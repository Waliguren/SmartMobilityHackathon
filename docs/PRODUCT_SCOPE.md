# Alcance Inicial

## Perfiles

### Gestor

- Consulta contratos activos
- Visualiza incidencias abiertas
- Planifica y reasigna visitas
- Supervisa carga por técnico
- Revisa reportes cerrados

### Técnico / instalador

- Consulta agenda asignada
- Accede al detalle de cada intervención
- Marca estados de ejecución
- Rellena reporte y sube evidencias

## MVP funcional previsto

### Operación base

- Login por rol
- CRUD básico de incidencias
- Listado y detalle de visitas
- Asignación manual de técnico
- Cierre de visita con reporte

### Datos operativos mínimos

- Estado de contrato
- Estado y prioridad de incidencia
- Fecha planificada de visita
- Duración estimada
- Ubicación de intervención
- Estado final y observaciones del reporte

## Fuera de alcance en esta fase

- Motor de optimización de rutas
- Recomendador automático de planificación
- Analítica avanzada o IA predictiva
- Integraciones con terceros
- Notificaciones push completas

## Criterio de esta base

Este repositorio solo deja:

- estructura
- documentación
- configuración inicial
- esqueletos técnicos mínimos

No deja casos de uso implementados.
